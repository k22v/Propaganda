# Code of Conduct

## Наши обязательства

Мы обязуемся создавать открытую и дружелюбную среду для всех участников.

## Стандарты поведения

- Уважительное отношение к другим участникам
- Конструктивная обратная связь
- Ориентация на решение проблем
- Профессиональное общение

## Обязанности участников

- Не допускать дискриминации
- Не допускать оскорбительного поведения
- Не публиковать спам

## Нарушение правил

Сообщите о нарушениях через Issues или лично администратору проекта.

## Применение

Администраторы проекта оставляют за собой право удалять или редактировать сообщения, нарушающие данный кодекс поведения.
